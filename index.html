Introduction

JavaScript is a versatile programming language used primarily for web development. It allows developers to create dynamic and interactive web applications.

  • JavaScript can be used for front-end development with frameworks like React, Angular, and Vue.
  • It is also used for back-end development with Node.js.
  • JavaScript is high-level and interpreted.
  • It supports dynamic typing.
  • Prototype-based object orientation is a key feature.
Syntax

The syntax of JavaScript is the set of rules that define a correctly structured JavaScript program. Here are some basic elements:

  • Statements: JavaScript programs are made up of statements.
  • Variables: Used to store data values.
  • Operators: Used to perform arithmetic on variables and values.
  • Functions: A block of code designed to perform a particular task.
  • Objects: Collections of properties and methods.

Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, and const.

The var keyword is used in older code and has function scope, while let and const are block-scoped.

Example:

var a = 10;
let b = 20;
const c = 30;
  • var is function-scoped and can be re-declared and updated.
  • let is block-scoped and can be updated but not re-declared within the same scope.
  • const is block-scoped and cannot be updated or re-declared.
  • Variables declared with var are hoisted to the top of their scope.
  • Variables declared with let and const are not initialized until their definition is evaluated.
Functions

Functions are blocks of code designed to perform particular tasks. They are executed when "called" (invoked).

A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().

Example:

function myFunction(p1, p2) {
return p1 * p2; // The function returns the product of p1 and p2
}
  • Functions allow you to reuse code.
  • Functions can take parameters and return values.
  • Functions can be defined using the function declaration or function expression syntax.
  • Arrow functions provide a shorter syntax and do not have their own this context.
  • Functions can be called with different numbers of arguments than they were declared to accept.
Events

JavaScript can "react" to events, like when a user clicks on a webpage element.

HTML events are "things" that happen to HTML elements. When JavaScript is used in HTML pages, JavaScript can "react" on these events.

Example:


function displayDate() {
document.getElementById("demo").innerHTML = Date();
}
  • Common events include clicks, mouse movements, and keyboard presses.
  • Event listeners can be added to HTML elements using the addEventListener method.
  • Events can be triggered manually using the dispatchEvent method.
  • Event propagation can be controlled using stopPropagation and preventDefault.
  • Event delegation is a technique for handling events efficiently.
